开放平台对接流程

搭建开放平台的核心是开放平台的业务能力，能够让第三方接入平台中。开放平台的设计一般涉及到交互协议、数据安全、平台测和商户测的平台开放的核心业务流程、平台接口设计四个主要环节。

一、申请成为开发者

小love容儿开放平台要求第三方接入用户请首先【注册】成为平台的用户,并通过主站的账号注册成为开放平台开发者。若您尚未注册平台账号，请先前往主站注册成为平台用户;

在开放平台点击【登录】进入主站登录页面，若未注册，请先在主站注册，公司、企业请使用法人信息注册，以免人员变换导致不变

登录成功后，点击右上角用户昵称下拉菜单的【个人资料】，进入开发者注册页面;

备注：如果当前登录账号未认证，会进入认证流程，个人、个体用户请申请个人认证，公司、企业请申请企业认证，只有认证通过后才能继续操作。【目前只支持企业认证，请一定要按照要求提供必要的资质证书】


	               			public class Thread1 implements Runnable {
							   Object lock;
							   public void run() {  
							       synchronized(lock){
							         ..do something
							       }
							   }
							}
